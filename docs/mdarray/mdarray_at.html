mdlib::mdarray<T, N...>::at

  constexpr reference at(expand<size_type, N>... pos);

  constexpr const_reference at(expand<size_type, N>... pos) const;

Returns a reference to the element at the specified indices, with bounds checking.


Parameters

pos - position of the element to return. An index must be provided for each dimension of the array.


Return value

Reference to the requested element.


Exceptions

std::out_of_range if an index is greater than number of elements in its dimension.


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    mdlib::mdarray<int, 3, 3> arr = {1, 2, 3,
                                     4, 5, 6,
                                     7, 8, 9};
    
    // Set elements
    arr.at(0, 1) = 22;
    arr.at(2, 0) = 121;
    
    try
    {
        // Out of bound access
        arr.at(0, 3) = 0;
    } 
    catch(const std::out_of_range& e) 
    {
        std::cout << e.what() << '\n';
    }

    // Print all values
    for(int i = 0; i < 3; ++i)
        for(int j = 0; j < 3; ++j)
            std::cout << arr.at(i, j) << ' ';
    std::cout << '\n';
}

Output

mdarray::at: pos (which is 3) >= dim (which is 3)
1 22 3 4 5 6 121 8 9
