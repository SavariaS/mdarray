std::get(xcontainer::mdarray)

  template<std::size_t... I, typename T, std::size_t... N>
  T& get(xcontainer::mdarray<T, N...>& arr);

Returns the Ith element from the multidimensional array.


Parameters

arr - the multidimensional array.


Return value

Reference to the requested element.


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    xcontainer::mdarray<int, 3, 3> arr = {1, 2, 3,
                                        4, 5, 6,
                                        7, 8, 9};

    std::cout << "arr[2][1] = " << std::get<2, 1>(arr) << '\n';
}

Output

arr[2][1] = 8
