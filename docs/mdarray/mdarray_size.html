mdlib::mdarray<T, N...>::size

  constexpr size_type size() const noexcept;

Returns the number of elements in the container.


Parameters

None.


Return value

The number of elements in the container.


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    mdlib::mdarray<int, 6, 2> arr = {2, 2,
                                        11, 24,
                                        30, 31,
                                        12, 4,
                                        67, 190,
                                        85, 31};
    

    // Iterate over the underlying array using the size of the container
    for(std::size_t i = 0; i < arr.size(); ++i)
    {
        std::cout << arr.data()[i] << ' ';
    }
    std::cout << '\n';
}

Output

2 2 11 24 30 31 12 4 67 190 85 31
