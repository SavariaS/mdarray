mdlib::mdarray<T, N...>::front

  constexpr reference front();

  constexpr const_reference front() const;

Returns a reference to the first element in the container.


Parameters

None.


Return value

Reference to the first element.


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    mdlib::mdarray<float, 2, 2> arr = {-1.0f, 0.0f,
                                        0.0f, 1.0f};
    
    std::cout << "The first number is: " << arr.front() << "\n";
}

Output

The first number is: -1
