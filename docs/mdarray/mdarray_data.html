mdlib::mdarray<T, N...>::data

  constexpr pointer data() noexcept;

  constexpr const_pointer data() const noexcept;

Returns pointer to the underlying one-dimensional array serving as element storage.


Parameters

None.


Return value

Pointer to the underlying element storage.


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    mdlib::mdarray<int, 3, 2, 2> arr = {1, 2,   3, 4,
                                        2, 4,   6, 8,
                                        1, 3,   5, 7};
    

    int* storage = arr.data();

    std::cout << "Data: ";
    for(int i = 0; i < arr.size(); ++i)
    {
        std::cout << storage[i] << ' ';
    }
    std::cout << '\n';
}

Output

Data: 1 2 3 4 2 4 6 8 1 3 5 7
