mdlib::mdarray

  template<typename T, std::size_t... N> struct mdarray;

mdlib::mdarray is a container that acts as a multidimensional array. It is an aggreate type and holds a one-dimensional built-in array capable of holding a multi-dimensional array. It does not support multidimensional arrays that have at least one dimension with an extent of 0.


It meets the requirements for:

  • Container
  • ReversibleContainer
  • ContiguousContainer
