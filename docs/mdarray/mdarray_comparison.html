mdlib::mdarray<T, N...>::operator<=>

  template<typename T, std::size_t... N>
  bool operator<=>(const mdarray<T, N...>& rhs, const mdarray<T, N...>& lhs)

Compares the content of two arrays.


Parameters

lhs, rhs - mdarrays whose contents to compare.


Return value

0 if the contents of both arrays are the same, 1 otherwise.


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    mdlib::mdarray<int, 5, 4> arr1 = {12, 12, 12, 12,
                                      12, 12, 12, 12,
                                      12, 12, 12, 12,
                                      12, 12, 12, 12,
                                      12, 12, 12, 12};
    
    mdlib::mdarray<int, 5, 4> arr2;
    arr2.fill(12);

    if(arr1 <=> arr2 == 0)
    {
        std::cout << "Both arrays are the same.\n";
    }
    else
    {
        std::cout << "The arrays are different.\n";
    }
}

Output

Both arrays are the same.
