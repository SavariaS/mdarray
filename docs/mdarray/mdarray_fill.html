xcontainer::mdarray<T, N...>::fill

  constexpr void fill(const value_type& value);

Assign the specified value to every element in the container.


Parameters

value - the value to be assigned to each element.


Return value

(none)


Example

#include <mdarray.hpp>
#include <iostream>

int main()
{
    xcontainer::mdarray<int, 4, 4> arr;
    arr.fill(2);
    
    for(int i = 0; i < 4; ++ i)
    {
        for(auto elem : arr[i])
        {
            std::cout << elem << ' ';
        }
        std::cout << '\n';
    }
}

Output

2 2 2 2 
2 2 2 2 
2 2 2 2 
2 2 2 2
