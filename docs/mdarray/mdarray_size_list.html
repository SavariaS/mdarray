xcontainer::mdarray<T, N...>::size_list

  constexpr size_ilist size_list() const noexcept;

Returns the number of elements in each dimension of the container.


Parameters

None.


Return value

The number of elements in each dimension.


Example

  #include <mdarray.hpp>
  #include <iostream>
  
  int main()
  {
      xcontainer::mdarray<int, 4, 8, 3> arr;
      
      // Print the size of every dimension
      std::cout << "The size of the array is: ";
      for(std::size_t dimension : arr.size_list())
      {
          std::cout << dimension << ' ';
      }
      std::cout << '\n';
  }

Output

The size of the array is: 4 8 3
