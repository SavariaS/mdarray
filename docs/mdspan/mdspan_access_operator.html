mdlib::mdspan<T, N...>::operator[]

  constexpr subspan operator[](size_type pos);

  constexpr const_subspan operator[](size_type pos) const;

Returns a subarray or a reference to the element at the specified position. No bounds checking is performed.


Parameters

pos - position of the element to return.


Return value

The subarray at the requested position if sizeof...(N) > 1

A reference to the requested element if sizeof...(N) == 1
